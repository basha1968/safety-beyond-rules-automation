# Safety Beyond Rules Automation

Automates sharing new posts from my Blogger blog "Safety Beyond Rules"[](https://safetybeyondrules.blogspot.com/) to LinkedIn, with plans to extend to YouTube video generation and Amazon affiliate links for safety products.

**Current Features (Starting Point):**
- Detects new blog posts via RSS feed: https://safetybeyondrules.blogspot.com/feeds/posts/default
- Auto-posts title, excerpt, link, and image to LinkedIn using GitHub Actions.

**Tech Stack:**
- GitHub Actions (free scheduled runs)
- RSS parsing
- LinkedIn API (via action)
